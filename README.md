# Raylib Game

Simple game built using Raylib in C++.

## Controls
- W / S / Up arrow / Down arrow: Move
- ESC: Exit
- ENTER: Proceed 

## Build
g++ main.cpp -o game -lraylib -lopengl32 -lgdi32 -lwinmm
